Phase 7 Onboarding

Bring your Spotify history into Spoolify

Local-first onboarding to validate your archive, import safely, and prepare for account sync when you are ready.

1. Request Spotify Data

Open Spotify Privacy settings and request your Extended Streaming History. Delivery usually takes a few days.

Expected files: one or more .json files containing playback events.

2. Validate Archive

Paste your downloaded file or folder path. Spoolify will sample entries, detect structure issues, and recommend import mode.

3. Import and Analyze

Run historical backfill first if this is your first import. Use sync prep when adding recent data before API sync.

Spotify Developer App Prep

Phase 9 will use your own Spotify Developer app. Plan these now:

  • Create app at developer.spotify.com
  • Set redirect URI for local callback (for example http://127.0.0.1:8787/callback)
  • Request scopes such as user-read-recently-played and user-read-currently-playing
